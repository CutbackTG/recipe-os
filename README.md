# Recipe OS

A modern, high-performance formulation, recipe, and specification platform
designed for large, multi-site food & beverage organisations.

## Goals
- Spreadsheet-speed formulation editing
- Instant cost, nutrition, and allergen impact
- Powerful search across 50k+ ingredients
- Enterprise-grade auditability and versioning
